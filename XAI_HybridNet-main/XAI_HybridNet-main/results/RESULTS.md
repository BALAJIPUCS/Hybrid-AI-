# Experimental Results

This directory contains the complete experimental evaluation and explainability outputs generated for the proposed **XAI-HybridNet** framework for interpretable pediatric pneumonia classification using chest X-ray images.

The results are organized into four major categories:

- **Performance Evaluation**
- **Explainability Visualizations**
- **Consensus and Stability Analysis**
- **Failure Case Analysis**

---

# Folder Structure

```text
results/
│
├── performance/
├── explainability/
├── consensus_stability/
├── failure_cases/
└── RESULTS.md
```

---

# Performance Evaluation

The `performance/` directory contains quantitative evaluation metrics, statistical benchmarking, calibration analysis, faithfulness evaluation, and training behavior visualizations for the proposed framework and baseline architectures.

## Included Analyses

- Classification performance comparison
- McNemar statistical significance testing
- Training and validation convergence behavior
- Confidence calibration evaluation
- Faithfulness deletion and insertion testing

---

# Explainability Visualizations

The `explainability/` directory contains multimodal interpretability outputs generated using CNN-based, Transformer-based, and model-agnostic explainability techniques.

## Included Explainability Methods

- Grad-CAM
- Guided Grad-CAM
- Vision Transformer Attention
- Hybrid Attention Fusion
- LIME Superpixel Analysis
- SHAP Attribution Analysis
- Multimodal Explanation Fusion

These visualizations demonstrate how the proposed XAI-HybridNet framework combines localized spatial attribution with global contextual reasoning for interpretable medical image analysis.

---

# Consensus and Stability Analysis

The `consensus_stability/` directory contains quantitative reliability evaluation of generated explanations using the proposed:

- **X-Consensus Score (XCS)**
- **X-Entropy Stability Metric**

The analysis evaluates:
- Cross-method explanation agreement
- Explanation consistency
- Per-sample explanation stability
- Attribution uncertainty

---

# Failure Case Analysis

The `failure_cases/` directory contains representative false-positive prediction examples generated by the proposed framework.

These cases highlight challenging pediatric chest radiographs where subtle pulmonary intensity variations and overlapping anatomical structures resemble pneumonia-related opacity patterns.

---

# Summary

The experimental results collectively demonstrate that the proposed **XAI-HybridNet** framework:

- Maintains competitive predictive performance
- Produces coherent multimodal explanations
- Achieves improved explanation consistency and stability
- Demonstrates reliable calibration behavior
- Supports interpretable AI-assisted medical image analysis

The repository is intended to provide transparent supplementary material supporting the experimental findings presented in the associated research manuscript.
