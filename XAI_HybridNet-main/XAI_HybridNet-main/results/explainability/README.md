# Explainability Visualizations

This folder contains multimodal explainability outputs generated using CNN-based, Transformer-based, and model-agnostic interpretability techniques within the proposed XAI-HybridNet framework.

## Included Figures

### gradcam_visualization.png
Grad-CAM heatmap visualization highlighting diagnostically important pulmonary regions identified by the CNN-based architecture.

### guided_gradcam_visualization.png
Guided Grad-CAM visualization combining gradient localization with fine-grained feature attribution for improved spatial interpretability.

### vit_attention_map.png
Vision Transformer attention visualization illustrating global contextual attention distribution across the chest radiograph.

### hybrid_attention_map.png
Hybrid attention visualization integrating CNN localization and Transformer contextual reasoning into a unified multimodal explanation.

### hybrid_multimodal_explanation.png
Integrated multimodal explanation output demonstrating combined explainability signals generated by the proposed HybridNet framework.

### lime_superpixel_explanation.png
LIME superpixel feature importance analysis comparing localized region importance distributions across ResNet-50, ViT-B/16, and HybridNet.

### shap_attribution_analysis.png
SHAP attribution visualization illustrating positive and negative feature contributions across image regions using model-agnostic explainability analysis.

### shap_region_attribution_comparison.png
Comparative SHAP region attribution analysis across ResNet-50, ViT-B/16, and HybridNet, demonstrating differences in attribution concentration and explanation consistency.
